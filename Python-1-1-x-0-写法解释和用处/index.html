Python [1, -1][x == 0] 写法解释和用处

发表于 2019-02-03 更新于 2021-03-24 分类于 Python 阅读次数：

解释

这种写法通俗形式为 [条件为假, 条件为真][判断条件]
在 Python 中，布尔型 True 转变为整数等于 1，False 转变为整数等于 0
所以条件为真时，返回第二个数，条件为假时，返回第一个数

用处

可以写成这样的形式
